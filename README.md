## Studi Kasus
Program ini merupakan aplikasi Program Rental Mobil, di mana pengguna dapat memilih jenis mobil (Avanza/Xenia/Innova), memasukkan durasi penyewaan, dan sistem akan menghitung total biaya rental secara otomatis.

## Fitur Program
- Pemilihan jenis mobil
- Input durasi sewa
- Perhitungan total biaya berdasarkan jenis mobil
- Perulangan untuk penyewaan lebih dari satu kali
